# 🔥 Elite Pronos - Application Automatique

Application web de pronostics sportifs avec système VIP, authentification Firebase et paiements Stripe.

## 🚀 Démarrage rapide

### Installation
\`\`\`bash
npm install
\`\`\`

### Configuration
\`\`\`bash
cp .env.example .env.local
# Édite .env.local avec tes clés Firebase, Stripe, etc.
\`\`\`

### Développement
\`\`\`bash
npm run dev
# Frontend: http://localhost:5173
# Backend: http://localhost:3000
\`\`\`

### Build & Production
\`\`\`bash
npm run build
npm start
\`\`\`

## 🔧 Technologies

- **Frontend**: React 18, Vite, TypeScript, Tailwind CSS
- **Backend**: Express, Firebase, Stripe
- **Auth**: Firebase Authentication + JWT
- **Database**: Firestore
- **Payments**: Stripe

## 📝 Variables d'environnement

Voir `.env.example` pour tous les paramètres requis.
